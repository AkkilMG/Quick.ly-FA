## Installation

```bash
git clone https://github.com/HeimanPictures/FastApi-Boiler
cd FastApi-Boiler
pip install -r requirements.txt
```

### Test

```bash
uvicorn main:app --reload  
```

### Production

```bash
uvicorn main:app --host
```
